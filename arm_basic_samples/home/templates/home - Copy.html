Azure Resource Manager Basic Samples

This Django based website enables you to try out the new Azure Resource Manager. Please make sure you have gone through the Settings tab to set basic authentication values.

Through this sample one can create a Virtual machine, storage account in a Virtual network within a resource group. You can eitehr create them individually, or create a complete deployment.

The next version will have more advanced options like building a complete Resource Manager template

Deployment option

This option will take you through a complete deployment, creating a resource group, adding a storage account and a virtual machine on a virtual network

Seperate Option (coming soon)

This option you can indvidually create resources

Template option (coming soon)

With this option you can generate your own custom ARM template
